谈谈vue-cli2-0和vue-cli3-0之间的差异

以前一直使用vue-cli2.0作为项目的脚手架开发，vue-cli3.0发布也有段时间了，但却没有使用过，借此对它做了一些了解，并谈谈3.0比之2.0有什么不同的差异。

1.从目录结构上面来看的话

2.0的目录结构如下：

3.0的目录结构如下：

从这里可以看到3.0和2.0比较，少了build、config等编译配置文件，这并不是移除掉了，而是让使用者更加注重功能开发，当然在3.0中你可以通过vueconf.js进行配置。

2.部分命令发生了变化

vue3.0是以下方式

  • 下载安装 npm install -g vue@cli
  • 删除了vue list
  • 创建项目 vue create
  • 启动项目 npm run serve

vue2.0

  • 下载安装 npm install -g vue@cli
  • 创建项目 vue init 项目类型(simple/webpack等) 自定义项目名
  • 启动项目 npm run dev
